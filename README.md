### 기능
- 서버 API에서 이미지를 받아와 화면에 나타낸다
- Pagination
- 사진 저장 / 삭제(FileManager, CoreData)

### 기술
- Code-base UI
- MVVM
- Data Binding (Observable)
- UICollectionViewLayout(Custom)
- URLSession
- CoreData, FileManager
- NSCache
